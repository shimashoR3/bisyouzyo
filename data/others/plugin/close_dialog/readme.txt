【ウィンドウクローズ時に確認ダイアログを表示するプラグイン】

■使い方
このテキストが入っているフォルダごと「data/others/plugin」フォルダに置きます。
それからfirst.ksとかに以下のように記述してください。
記述した時点から確認画面表示が有効になります。

[plugin name=close_dialog]
指定可能属性
text：確認ダイアログに表示するメッセージを指定
ok：確認ダイアログの「OK」ボタンに表示するテキストを指定
cancel：確認ダイアログの「キャンセル」ボタンに表示するテキストを指定


■できること
・ゲームウィンドウを閉じようとしたとき、またはリロードしようとしたとき、確認ダイアログを表示する

※ダイアログUIはアプリケーションorブラウザのデフォルトのものになります。ティラノスクリプトの[dialog]タグで使用できるものとは異なります。
※ティラノスタジオでのテストプレイ時にこのプラグインを使用すると、ゲーム画面をリロードするたびに確認ダイアログが表示されます。
※ブラウザプレイ時は、プラグインの動作前にユーザーの操作（タップなど）が必要です。ユーザーが最初に画面をタップしてからプラグインが有効になります。
※一部ブラウザで動作時は、確認ダイアログのメッセージ指定等は反映されません。ブラウザ規定のダイアログが表示されます。
